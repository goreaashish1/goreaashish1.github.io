Internet

The internet is a massive mesh of inter-connected devices implemented on a global scale and is primarily used to search, access and retrieve information.

WWW

The information accessed through the internet is in the world wide web which is an ever increasing collection of content.

Hypertext

Hypertext is a software system used to link the content on the screen to some related content which is accessed by clicking on the link.

Multimedia

The use of more than one medium to convey relevant information is called multimedia.

CLI

It stands for Command Line Interface, and is a way to communicate with a device by using a set of precoded instructions or commands.

GUI

The Graphical User Interface has the same function as that of a CLI but, it is achieved by use of indicators such as buttons, icons, windows, menus etc.

HCI

Human Computer Interaction is the study of actions and reactions of humans towards machines and the extent to which a machine is developed (or can be developed) so as to make it easier for humans to use.

FTP

It stands for File Transfer Protocol, and is the standard way to transmit files by use of TCP/IP connection and it requies authentication from both sides.

SSH

It stands for Secure Shell, and is a network protocol that allows secure access to a remote machine on a different network.

HTTP

It stands for Hyper-Text Transfer Protocol. It is a way of transmitting files (text, images, videos) to and from the world wide web over the internet.

TELNET

It is a protocol that allows a secure access to a machine on the same network.

HTML

It stands for Hyper-Text Markup Language. It is universal standard for marking up content of a document into relevant tags for better handling of content on the web page.

CSS

It stands for Cascading Style Sheets, it is a way of formatting and laying out marked-up content with the use of designing elements.

DTD

It stands for Data Type Definition, and is usually the first tag that we write (or forget to write) while marking up content of a web page in editor.
